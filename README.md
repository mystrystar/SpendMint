# SpendMint 💸🪙

**Turn subscription waste into wealth.**

SpendMint is an AI-powered subscription tracker that helps users monitor recurring expenses, detect unused subscriptions, and convert saved money into simulated gold, silver, or cash savings.

## Features

* 📌 Track all subscriptions in one place
* 🤖 Detect underused or wasteful subscriptions
* 💰 Pause/cancel and redirect savings into a wealth vault
* 🪙 Simulated Gold & Silver savings tracker
* 📚 Read-to-Earn financial literacy rewards
* 📊 Analytics for spend, savings, and wealth growth
* 🔔 Smart renewal reminders

## Tech Stack

* React + Vite + TypeScript
* TailwindCSS + shadcn/ui
* Neon PostgreSQL
* Drizzle ORM
* TanStack Query
* Netlify

## Vision

Most subscription trackers stop at reminders.
SpendMint goes further by helping users **cut waste, build savings, and develop better wealth habits.**

**Save smarter. Grow better.**
